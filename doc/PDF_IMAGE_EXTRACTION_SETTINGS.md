# PDF 이미지 추출 설정 가이드 (Extraction Settings)

이 문서는 `scripts/extract_images.py` 스크립트의 이미지 추출 필터링 기준을 정의하고 수정하는 방법을 설명합니다.
불필요한 아이콘, 장식, 머리글/바닥글 이미지를 제외하고 **의미 있는 삽화(Figure/Illustration)**만 추출하기 위해 설정을 조정할 수 있습니다.

## 1. 주요 설정 변수 (Configuration)

스크립트(`scripts/extract_images.py`) 상단의 `CONFIG` 딕셔너리에서 아래 값들을 변경하세요.

| 변수명 | 기본값 | 설명 |
| :--- | :--- | :--- |
| `MIN_WIDTH` | 100 | 이미지의 최소 가로 픽셀 크기입니다. 이보다 작은 이미지는 무시합니다. |
| `MIN_HEIGHT` | 100 | 이미지의 최소 세로 픽셀 크기입니다. 이보다 작은 이미지는 무시합니다. |
| `MIN_FILE_SIZE_KB` | 5 | 파일 용량(KB) 최소값입니다. 너무 용량이 작은(단순 패턴 등) 이미지를 거릅니다. |
| `ASPECT_RATIO_LIMIT` | 5.0 | 가로/세로 비율이 너무 극단적인 이미지(긴 막대, 선 등)를 제외합니다. (예: 5.0이면 가로가 세로보다 5배 긴 경우 제외) |
| `EXCLUDE_EXTENSIONS` | `['jb2']` | 제외할 파일 확장자 목록입니다. |

## 2. 필터링 조정 팁 (Tuning Tips)

### 상황 A: 너무 작은 아이콘들이 자꾸 추출될 때
*   `MIN_WIDTH`와 `MIN_HEIGHT`를 **200** 또는 **300**으로 높이세요.
*   의미 있는 삽화는 보통 300px 이상인 경우가 많습니다.

### 상황 B: 페이지 배경이나 테두리 선이 추출될 때
*   **파일 용량** 필터를 사용하세요. 배경 패턴은 픽셀 수는 커도 용량(KB)은 매우 작을 수 있습니다. (`MIN_FILE_SIZE_KB`를 10~20으로 상향)
*   **비율(Aspect Ratio)** 제한을 둡니다. 테두리 선은 가로는 매우 길고 세로는 짧으므로 `ASPECT_RATIO_LIMIT`에 걸리게 됩니다.

### 상황 C: 특정 페이지의 이미지만 다시 뽑고 싶을 때
*   스크립트 실행 시 인자로 페이지 범위를 주거나, 코드를 수정하여 특정 페이지만 타겟팅할 수 있습니다.

## 3. 실행 방법

```bash
# 기본 설정으로 실행
python scripts/extract_images.py "파일경로.pdf"

# 설정 변경 후 실행
# (스크립트 파일을 열어 상단의 CONFIG 값을 수정한 뒤 저장하고 실행하세요)
```
